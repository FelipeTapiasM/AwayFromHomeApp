# AwayFromHomeApp
Aplicación móvil para gestión de tiempo y finanzas de estudiantes foráneos

## Tecnologías
- React Native
- Expo
- React Navigation
- React Native Paper

## Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/AwayFromHomeApp.git

# Instalar dependencias
npm install

# Ejecutar proyecto
npm start
